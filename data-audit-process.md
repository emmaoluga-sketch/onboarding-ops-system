# Data Audit Process

## Step 1: Data Collection
- Export data from all systems (Deel, BambooHR, Sheets)

## Step 2: Field Mapping
- Align fields:
  - Name
  - Role
  - Compensation
  - Tax data

## Step 3: Error Detection
- Missing values
- Duplicate entries
- Mismatched compensation

## Step 4: Cleanup
- Remove duplicates
- Update stale records
- Validate contracts

## Step 5: Validation
- Cross-check with source documents
