# Onboarding Playbook

## Objective
Ensure a consistent, efficient onboarding experience for all new hires.

## Principles
- No missing documentation before Day 1
- Full system access within 24 hours
- Clear ownership between HR and Manager

## Stakeholders
- HR: Documentation + coordination
- Manager: Role onboarding
- IT: System access

## Success Metrics
- Time-to-productivity
- Completion rate
- Employee satisfaction
